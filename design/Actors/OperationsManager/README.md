# Operations Manager
The Operation Manager is responsible for managing the operations of the system.
This includes the creation and management of environments, users, and connectivity to the Clouds.

## Use Cases
* Manage Policy
* Manage Applications
* Plan Capacity
* Manage Users
* Manage Clouds
* Map Cloud Resources
